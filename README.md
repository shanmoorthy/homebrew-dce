This is a Homebrew tap repository that provides the `dce` formula for Disposable Cloud Environment (DCE).

## Install for macOS
### Homebrew
To install:
```sh
brew install shanmoorthy/dce/dce
```
To upgrade:
```sh
brew upgrade dce
```
